変数型 「int (入れ物の名前);」などとして変数を宣言
  • int 整数型。整数ですな
  • double　浮動小数点型倍精度。まあ実数だとおもっておけば・・
  • float　浮動小数点型。まぁ実数ですな
  • char　文字1個
  • string 文字列（ただし、これは#includeを追加せねばならない
    using namespace std;とはじめにかかないとつかえない人
  • cout cout << "文字列";出力するシーアウト
  • cin cin >> 変数;入力した情報を取り込む。
  • endlendlとうつと改行
    論理演算子、ifとかforとかにつかってあげてください
  • != 両辺の値が違うときに1,イコールなときに0
  • >,< 不等号がただしいときに1,正しくないと0
  • == 両辺がイコールなら1,そうでなければ0
  • <=.>= いわゆる以上以下
  • && and演算子、両辺が真なら1それいがいは0
  • || or演算子、両辺とも偽なら0それ以外は1
    ループとか条件分岐。クリックすると詳しく説明
  • for for(int i=0;i<10;i++){1ループの操作}
  • while while(ここを判定して=1なら続ける){１ループの操作}
  • switch switch(ここを判定して){ case x: 判定した値がxの時の操作　break; case y: break;default}
  • if if(ここを判定して){1ループの操作}
  • else ifの条件を満たさなかったときの操作
  • break ループから抜ける
